# GitHub Pages Activation Guide

## For: dropship-blueprint-dashboard

### Step 1: Navigate to Settings

1. Go to: https://github.com/Purmamarca/dropship-blueprint-dashboard
2. Click the **"Settings"** tab (top right of the repository page)

### Step 2: Find Pages Section

1. In the left sidebar, scroll down to find **"Pages"**
2. Click on **"Pages"**

### Step 3: Configure GitHub Pages

You'll see a section titled **"Build and deployment"**

#### Source:

- Click the dropdown under "Source"
- Select: **"Deploy from a branch"**

#### Branch:

- Click the first dropdown (currently shows "None")
- Select: **"main"**
- Click the second dropdown (folder)
- Select: **"/ (root)"**

### Step 4: Save

- Click the **"Save"** button

### Step 5: Wait for Deployment

- GitHub will show a blue info box saying "GitHub Pages source saved"
- Wait 1-2 minutes for the build to complete
- Refresh the page
- You'll see a green success message with your live URL:
  **"Your site is live at https://purmamarca.github.io/dropship-blueprint-dashboard/"**

---

## Verification:

Once you see the green success message, your site is live!

Visit: https://purmamarca.github.io/dropship-blueprint-dashboard/


A premium, cinematic web dashboard providing a high-intent roadmap and financial lattice for scaling dropshipping businesses.

## Features
- **Advanced Financial Logic:** Real-time Break-even ROAS and Net Margin calculations.
- **Premium Design:** Glassmorphic UI with smooth animations.
- **Actionable Roadmap:** 8-step blueprint from niche selection to scaling.

## Tech Stack
- Vanilla HTML5 / CSS3
- JavaScript (ES6+)
- Lucide Icons
